# Tracking File Retention Policy

## Purpose
Define operator-visible retention windows for generated files and mandatory handling guidance.

## Required Warning Text
The generation completion panel must communicate:

- `Please download and securely save:`
  - Labels PDF
  - Money Orders
  - Tracking Master File
- `Files are automatically removed from server after retention period.`

## Retention Windows
- FREE users: 24 hours
- PAID users: 72 hours

## Operator Requirements
1. Download required artifacts immediately after generation completes.
2. Store files in approved internal storage before retention deadline.
3. Do not rely on server copies after retention cutoff.
4. Use Tracking Batch History actions for reruns while source is still retained.
