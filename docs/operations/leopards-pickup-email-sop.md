# Leopards Pickup Email SOP (Future Phase)

## Status
Planned for future phase. Not active in Phase 1.

## Required Payload
- Customer name
- Customer phone
- Pickup address
- Pickup city
- Number of articles
- Total bundle weight
- Destination hub (Lahore or Sahiwal)
- Service type
- Special notes
- Internal booking ID
- Secure status update link

## Controls
- Signed expiring update links
- Send retry policy
- Bounce handling
- Audit log of sent/resend events
